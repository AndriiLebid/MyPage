## Deploy
This site is hosted with **GitHub Pages**.
- Main entry file: `index.html` (root)
- Deployed from the `main` branch via repository Settings → Pages  
(GitHub Pages supports publishing a website from a repository.) [4](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site/verifying-your-custom-domain-for-github-pages)[3](https://wiki.zahno.dev/tools/cloudflare/gh-pages.html)
